# Интернет-магазин мебели "Интерьер"

Адаптивный сайт интернет-магазина мебели с каталогом товаров, корзиной и формой оформления заказа.

## 📋 О проекте

Проект представляет собой многостраничный сайт для мебельного магазина с удобной навигацией, фильтрацией товаров и адаптивным дизайном для всех устройств.

### Страницы сайта:
- **Главная** (`index.html`) - приветственный экран, категории мебели, специальные предложения
- **Каталог** (`catalog.html`) - полный список товаров с фильтрацией
- **Корзина** (`cart.html`) - выбранные товары и оформление заказа

## 🛠 Технологии

- HTML5
- SCSS (разделен на модули)
- CSS3
- Адаптивная верстка (Desktop First)
- БЭМ-методология

## 🎨 Особенности дизайна

- Современный минималистичный дизайн
- Адаптация под мобильные устройства (320px)
- Планшетная версия (1024px)
- Кастомные элементы форм (радио-кнопки, чекбоксы, селекты)
- Двойные ползунки для фильтрации по цене
- Фильтрация по цветам, типу комнаты, срокам доставки

## 🚀 Функциональность

### Каталог товаров
- Сетка товаров 3xN на десктопе
- Адаптивная сетка (3 → 2 → 1 колонка)
- Фильтрация товаров по:
  - Срокам доставки
  - Цене (двойной ползунок)
  - Цвету (14 цветов)
  - Типу комнаты (10 категорий)

### Корзина
- Список выбранных товаров
- Изменение количества
- Промо-блок "Специальные предложения"
- Форма оформления заказа
- Расчет итоговой суммы

### Адаптивность
- **Desktop** (>1024px) - полная версия
- **Tablet** (≤1024px) - оптимизированная верстка
- **Mobile** (≤767px) - мобильная версия с выпадающим меню

## 🎯 Особенности верстки

- Flexbox и Grid для построения сеток
- Кастомные стили для форм
- Плавные анимации при наведении
- Оптимизация изображений для мобильных
- Семантическая верстка
